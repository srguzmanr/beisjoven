Header anatomy

Components stacked in order

All three bands share the same stack. Only the brand row changes between variants. Mobile auto-hides on scroll down, reveals on scroll up (80px threshold). Desktop shrinks to the nav row on sticky.

1. Brand row
Mark-only mobile · Mark+Wordmark desktop · 44×44 tap targets
2. Nav
Mobile: horizontal scrollable chips w/ right-edge fade gradient · Desktop: inline category list w/ 2px underline on active
3. Quick Hits
Always-present, admin-editable, rotates every 4.5s, 36px tall
4. LIVE strip
Conditional (event-only). Sits ABOVE Quick Hits when active. Red EN VIVO pill + pulse, category color tab
Search
Icon-only (44×44, aria-label="Buscar") on all breakpoints. Tap opens full-viewport overlay
